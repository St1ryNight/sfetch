# sfetch
A neofetch clone written in Rust supporting Macos, Linux, Windows, FreeBSD, OpenBSD, and Illumos

![img alt](https://github.com/St1ryNight/sfetch/blob/4fcac7c4b53ce72f0fb068ef48c66fccf59d6bd1/screenshot.png)
## Requirements
[+] Cargo

## Compile
```sh
git clone https://github.com/St1ryNight/sfetch.git && cd msfetch/
cargo run (executable lives in target/debug/
```
## read me 
Linux, Windows, and Illumos are missing some things that will be added eventually
