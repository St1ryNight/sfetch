# sfetch
A neofetch clone written in Rust supporting Macos, freebsd and openbsd
![img alt](https://github.com/St1ryNight/sfetch/blob/7cb27ea8264f571360591bfbba4a5f08b80bb0a6/screenshot.png)
## Requirements
[+] Cargo

## Compile
```sh
git clone https://github.com/St1ryNight/sfetch.git && cd msfetch/
cargo run (executable lives in target/debug/
```
## read me 
Linux, Windows, and Illumos are missing some things that will be added eventually
